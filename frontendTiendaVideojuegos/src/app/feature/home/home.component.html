> Esta aplicación web cuenta con los siguientes servicios:


Cliente
Crear las credenciales

Se puede crear un usuario con sus datos personales, para poder realizar una reserva.

Cliente
Eliminar las credenciales

Si el cliente desea no pertenecer más a la tienda, podrá eliminar su usuario cuando deseé.

Videojuego
Agregar juego

Se puede crear un juego para ser reservado inmediatamente, siempre y cuando cumpla las reglas de negocio.

Videojuego
Reglas de negocio

No se puede crear un juego los domingos, el nombre del juego no puede contener los siguientes caracteres (*, /, +).

Reserva
Reservar un juego

Podrás reservar cualquier juego, a la hora que desees, siempre y cuando tengas un usuario creado y exista el juego que deseas.
